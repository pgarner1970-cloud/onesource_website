# One Source Air & Energy Ltd - Sample Static Website

This is a hostable static sample site.

## Uploading

Upload the contents of this folder to your web hosting public folder, for example:

- `public_html/`
- `htdocs/`
- `www/`

The homepage file is:

- `index.html`

## Structure

- `index.html` - main homepage
- `assets/css/styles.css` - all styling
- `assets/js/site.js` - mobile menu and gallery filtering
- `assets/images/` - placeholder SVG images

## Replacing images

The current images are simple UK/domestic style SVG placeholders.

Replace files in `assets/images/` with real project photos later. Keep the same filenames if you do not want to edit the HTML.

## Backend later

This static version can later be extended with:

- PHP/MySQL project gallery admin
- WordPress custom post types
- Headless CMS gallery
- Simple JSON/project file editor
